# RoboMaster 2026 战术沙盘

这是一个零依赖的本地单页工具，直接打开 `index.html` 就能用。

## 已实现

- 按队伍放置兵种
- 拖拽调整兵种站位
- 以选中对象为起点绘制路线
- 修改名称、图标、颜色、备注、尺寸
- 一键生成红/蓝方标准阵容
- 支持把当前摆位、路线和时间轴保存成多个可复用战术模式
- 大地图沙盘只显示 1 / 2 / 3 / 4 / 哨，并支持 5 个自定义图标插槽
- 内置 7 分钟比赛时间轴，标出能量机关、飞镖、前哨/堡垒、哨兵补给、空中支援时间等关键节点
- 补充场地增益与装配规则卡，写明基地/高地/前哨/堡垒/补给/装配区/地形跨越的触发方向、条件和数值
- 导入地图底图
- 本地保存、JSON 导入导出
- 支持离线包默认方案，别人解压后双击 `index.html` 就能直接看到内置内容
- 支持从当前页面一键导出专属离线包，能把你自己的底图、模式库、路线一起打进 zip，别人可在此基础上继续修改并再次导出

## 规则预设依据

我从当前目录里的 `RoboMaster 2026 机甲大师超级对抗赛比赛规则手册V1.4.1（20260417）.pdf` 提取了几个直接影响沙盘建模的默认值：

- 战场默认比例为 `28m x 15m`
- 当前沙盘主视图默认聚焦：英雄、工程、步兵 `2`、哨兵
- 标准阵容按钮默认生成：英雄 `1`、工程 `1`、步兵 `2`、哨兵 `1`

## 使用方式

1. 打开 `index.html`
2. 从左侧兵种模板选择阵营并点击沙盘放置
3. 选中对象后拖动位置，或在右侧修改属性
4. 点击“开始绘线”后，在沙盘上逐点添加路线
5. 用“导出方案”保存推演结果，或用“导入方案”继续编辑

## 离线分发

- 将 `index.html`、`app.js`、`styles.css`、`bundle-state.js`、`bundle-source.js` 一起打包发给对方即可
- 对方第一次打开时，会自动加载压缩包内置的默认方案
- 如果对方改动后想恢复到压缩包自带版本，可以点击页面顶部的“恢复离线包默认方案”
- 如果你想把“你当前页面里的真实方案”直接发给别人，刷新后点击页面顶部的“导出当前离线包”，下载出的 zip 会带上底图、模式、路线、时间轴状态和完整页面文件
- 对方必须先解压 zip，再打开解压后的 `index.html`；不要直接在压缩包预览窗口里双击打开
- 解压后可以继续摆放、画线、保存模式，并再次点击“导出当前离线包”生成新的 zip
- 当前版本已把“原项目页”和“导出的单文件离线页”本地缓存隔离，避免两边互相覆盖浏览器里的沙盘数据
